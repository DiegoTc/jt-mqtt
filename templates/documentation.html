Quick Start Scripts

Automated scripts to help you quickly set up and test the system.

    Interactive script to start all components for testing

    ./start_test_environment.sh
  • Monitor MQTT Traffic

    Command to subscribe to all PetTracker topics

    mosquitto_sub -t "pettracker/#" -v
  • Start MQTT Broker

    Command to start the Mosquitto broker

    mosquitto -c mosquitto.conf
